# G-minis-project
**Programa de control de inventario para la papeleria**

Resumen del proyecto – Implementación de un programa de control de inventarios para Papelería G-minis

La Papelería G-minis, ubicada en Ixtapaluca, Estado de México, es un negocio familiar con más de 15 años ofreciendo artículos escolares, servicios de copiado, fax y apoyo escolar. Con el paso del tiempo, el negocio se ha consolidado como un punto clave dentro de la zona habitacional, ampliando su catálogo de productos y servicios.

A pesar de su crecimiento, la papelería nunca ha contado con procesos sistematizados debido a las limitaciones económicas comunes en pequeños negocios. Sin embargo, tras analizar la operación actual y dialogar con los dueños, se decidió implementar un programa de control de inventarios que permita mejorar la administración diaria del negocio.

El objetivo del proyecto es desarrollar un software sencillo y funcional, capaz de registrar ventas por producto, manejar inventarios en tiempo real y calcular de manera automática el costo, el precio de venta y el margen de ganancia de cada artículo. Con esto, se busca eliminar errores asociados a registros manuales, facilitar los cortes diarios y ofrecer una visión clara de la rentabilidad individual de cada producto o servicio. Además, el programa permitirá conocer las existencias actualizadas, lo que ayudará a realizar compras basadas en datos y evitar tanto el desabasto como la acumulación innecesaria de inventario.

Como parte de los entregables del proyecto, se incluirán:
•	Documentos de aprobación por fase, validados junto con los dueños, para asegurar que el desarrollo se mantenga dentro del alcance acordado.
•	Documento funcional del programa, detallando los datos capturados, campos editables, cálculos integrados y estructura general del sistema.

Este proyecto representa un paso importante hacia la modernización del negocio, optimizando la gestión diaria y brindando información precisa para tomar decisiones financieras más acertadas.
